# 🏹 Hunter's Path — как запустить приложение (Путь А)

Это приложение: карта охоты + **живой чат с Крестодателем внутри**. Прогресс сохраняется, открывается с телефона и компа, можно повесить иконку на домашний экран.

Чтобы чат заработал, нужно один раз настроить (≈30 минут). Делается в 3 этапа. Иди строго по шагам.

---

## ЭТАП 1. Получить ключ Claude API (5–10 минут)

Ключ — это «пропуск», по которому приложение обращается к Claude. Хранится он на сервере, не в браузере (безопасно).

1. Зайди на **https://console.anthropic.com**
2. Зарегистрируйся (тот же email, что и в Claude — удобно).
3. Слева открой раздел **Billing** → пополни баланс (минимум $5 хватит надолго; на твоих объёмах чат стоит копейки).
4. Слева открой **API Keys** → кнопка **Create Key** → назови «hunters-path» → скопируй ключ.
   - ⚠️ Ключ показывается ОДИН раз. Скопируй и сохрани временно в заметку. Начинается с `sk-ant-...`

---

## ЭТАП 2. Залить приложение на Vercel (10–15 минут)

Vercel — бесплатный хостинг. Деплой в пару кликов.

### 2.1. Создай аккаунт
1. Зайди на **https://vercel.com** → **Sign Up** → войди через **GitHub** (если нет GitHub — создай на github.com, 2 минуты).

### 2.2. Залей код
Самый простой способ — через сайт, без терминала:

1. Зайди на **https://github.com** → **New repository** → имя `hunters-path` → **Create**.
2. На странице репозитория нажми **uploading an existing file**.
3. Перетащи туда ВСЕ файлы из папки `hunters-path` (включая папки `api` и `public`). Структура должна остаться такой же:
   ```
   hunters-path/
   ├── api/
   │   └── chat.js
   ├── public/
   │   ├── index.html
   │   ├── manifest.json
   │   ├── icon-192.png
   │   └── icon-512.png
   ├── vercel.json
   └── package.json
   ```
4. Нажми **Commit changes**.

### 2.3. Подключи к Vercel
1. На **vercel.com** → **Add New → Project**.
2. Выбери репозиторий `hunters-path` → **Import**.
3. **ВАЖНЫЙ ШАГ:** перед деплоем раскрой **Environment Variables** и добавь:
   - **Name:** `ANTHROPIC_API_KEY`
   - **Value:** вставь свой ключ `sk-ant-...`
   - Нажми **Add**.
4. Нажми **Deploy**. Подожди минуту.
5. Готово — Vercel даст тебе ссылку вида `https://hunters-path-xxx.vercel.app`

---

## ЭТАП 3. Повесить на домашний экран (2 минуты)

### На iPhone:
1. Открой ссылку в **Safari**.
2. Нажми кнопку «Поделиться» (квадрат со стрелкой) → **На экран «Домой»**.
3. Иконка 🏹 появится как приложение.

### На Android:
1. Открой ссылку в **Chrome**.
2. Меню (⋮) → **Добавить на главный экран**.

---

## ✅ Проверка
- Открой приложение → вкладка **Крестодатель** → напиши «привет».
- Если ответил — всё работает, ключ настроен правильно.
- Если ошибка про ключ — вернись в Vercel → Settings → Environment Variables, проверь `ANTHROPIC_API_KEY`, и нажми **Redeploy**.

---

## Что внутри
- **Карта охоты** — 6 этапов, свободный выбор квестов, боссы под замком, трофеи и празднование.
- **Чат Крестодателя** — живой Claude, знает все твои цифры, команду и конвейер. Кнопка «помощь» у квеста кидает запрос прямо в чат.
- **Прогресс** сохраняется в браузере устройства.

## Мелочи
- Чтобы менять/добавлять квесты — правь массив `STAGES` в `public/index.html`.
- Характер и данные Крестодателя — в `api/chat.js` (переменная `SYSTEM`). Можешь дополнять контекст.
- Прогресс хранится локально на устройстве. Хочешь облачную синхронизацию между телефоном и компом — это следующий апгрейд (нужна БД), скажи Крестодателю в чате.
